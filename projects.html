DRIVE: Directed Remote Interactive Viewing and Exploring

Android video chat app for giving and recieving virtual tours

School Project for RWU

September 2021 - May 2022

DRIVE is a video-chat Android application designed specifically for giving and receiving virtual tours. The client for this project was Dr. Sonya Cates, a professor of computer science at RWU. Close corporation was required between the development team and the client for the development of DRIVE.

I was responsible for both the front and backend of this project. This includes user interface design, server-side design, and NoSQL database design.

Skills Involved: Javascript, React-Native, Google Firebase, NoSQL, Socket.IO, NodeJS

Portfoilio Website (This Website)

A static website dispalying my resume, skills, school/work projects, and my contact information.

Personal Project

November 2022

This static website was written in pure HTML and CSS, with no framework. The website files are stored in an publicly accessable AWS S3 bucket. AWS CloudFront is used to cache the website files and quickly serve them with low latency. The website uses the HTTPS protocol.

The "Send me a message" section on the home page uses AWS SES to send emails to me at rudolphdesanti@gmail.com. Through an HTML API made with AWS API Gateway, data os transfered to an AWS Lambda serverless function, where AWS SES is used to send the emails.

Skills Involved: HTML, CSS, Javascript, AWS CloudFront, AWS Simple Storage Service (S3), AWS Lambda, AWS Simple Email Service (SES), AWS API Gateway, AWS IAM policies & roles
